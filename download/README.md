### Download Instructions

Place the downloaded source in this directory, like:

- binutils-2.24.tar.bz2
- expat-2.0.1.tar.gz
- gcc-4.8.2.tar.bz2
- gdb-7.6.2.tar.bz2
- gmp-5.0.4.tar.bz2
- mpc-1.0.1.tar.gz
- mpfr-2.4.2.tar.bz2
- newlib-1.20.0.tar.gz
- zlib-1.2.3.tar.bz2

... but it's easier to edit versions in '_env-only.sh' and run 'auto.sh' in root dir.
